# Tencent Location JavaScript API V2 Type Definitions 

## Background

这是一份 [腾讯位置服务 - JavaScript API](http://lbs.qq.com/javascript_v2/doc/) V2 的Type Definitions 文件。

主要是为现代化的前端项目在异步加载，引用腾讯地图JS API的时候，能够给现代化的IDE提供比较完整的类型定义和类型推导。
